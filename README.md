## Overview

This repository contains the deliverables for **Task 3: Web Application Security** completed as part of the Cybersecurity Internship at ApexPlanet Software Pvt. Ltd.

The task focuses on identifying, exploiting, and understanding common web application vulnerabilities using **DVWA (Damn Vulnerable Web Application)** in a controlled laboratory environment. Practical exercises were performed to understand the impact of these vulnerabilities and the security measures used to mitigate them.

---

## Objectives

* Perform SQL Injection attacks and database enumeration.
* Demonstrate Reflected and Stored Cross-Site Scripting (XSS).
* Perform Cross-Site Request Forgery (CSRF) attacks.
* Demonstrate Local and Remote File Inclusion.
* Intercept and analyze HTTP requests using Burp Suite.
* Analyze HTTP Security Headers.
* Understand secure coding practices and mitigation techniques.

## Activities Performed

### 1. SQL Injection

* Database Initialization
* SQL Injection Testing
* User Enumeration
* Database Version Detection
* Database Name Enumeration
* Table Enumeration
* Password Hash Extraction
* SQL Injection Prevention

### 2. Cross-Site Scripting (XSS)

* Reflected XSS
* Stored XSS
* Input Validation and Mitigation

### 3. Cross-Site Request Forgery (CSRF)

* Password Change Attack
* Authentication Verification
* CSRF Protection Techniques

### 4. File Inclusion

* Local File Inclusion (LFI)
* Remote File Inclusion (RFI) Attempt
* File Inclusion Mitigation

### 5. Burp Suite Advanced

* HTTP Request Interception
* Request Modification
* Burp Intruder Configuration
* Payload Testing

### 6. Security Headers Analysis

* HTTP Header Inspection using curl
* SecurityHeaders.com Analysis
* Review of Missing Security Headers
